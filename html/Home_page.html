desarrollo web

Un desarrollador web se encarga de crear páginas web utilizando varios lenguajes de programación. Sus responsabilidades incluyen el diseño y la programación de la funcionalidad del sitio web, cuyo fin es el de satisfacer las necesidades de los usuarios. Para llevar a cabo estas tareas, los desarrolladores deben tener un conjunto de habilidades en diferentes especialidades, por ello, es común escuchar los términos: front-end, back-end y full-stack.

front-end

Un desarrollador front-end se encarga de construir o programar todo lo que se puede visualizar en el navegador web, es decir, es responsable del diseño de la página web, su funcionalidad y otros elementos del lado del cliente. Estos desarrolladores deben dominar los siguientes lenguajes de programación: HTML, CSS y JavaScript

  • HTML
  • CSS
  • JavaScript
